Introducción a los servicios de red de Azure

Los servicios de red en Azure proporcionan una variedad de capacidades de red que se pueden usar juntas o por separado.

Servicios de conectividad

Conecte los recursos de Azure y los recursos locales mediante cualquiera o una combinación de estos servicios de red en Azure: red virtual (VNet), WAN virtual, ExpressRoute, puerta de enlace VPN, puerta de enlace NAT de red virtual.

Servicios de protección de aplicaciones

Proteja sus aplicaciones usando cualquiera de estos servicios de red o una combinación de ellos en Azure: Load Balancer, Private Link, protección DDoS, Firewall, Network Security Groups, Web Application Firewall y Virtual Network Endpoints.

Servicios de entrega de aplicaciones

entregue aplicaciones en la red de Azure utilizando cualquiera o una combinación de estos servicios de red en Azure: Red de entrega de contenido (CDN), Azure Front Door Service, Traffic Manager, Application Gateway, Internet Analyzer y Load Balancer.

Supervisión de red

Supervise los recursos de su red mediante cualquiera de estos servicios de red o una combinación de ellos en Azure: Network Watcher, ExpressRoute Monitor, Azure Monitor o VNet Terminal Access Point (TAP).
