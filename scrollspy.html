  • Mas opciones
    • Application Gateway
    • Azure Load Balancer

Front Door

Es una red de entrega de aplicaciones que proporciona equilibrio de carga global y servicio de aceleración de sitios para aplicaciones web. Ofrece capacidades de capa 7 para su aplicación, como descarga de SSL, enrutamiento basado en ruta, conmutación por error rápida, almacenamiento en caché, etc. para mejorar el rendimiento y la alta disponibilidad de sus aplicaciones.

Traffic Manager

Es un equilibrador de carga de tráfico basado en DNS que le permite distribuir el tráfico de manera óptima a los servicios en las regiones globales de Azure, al tiempo que proporciona alta disponibilidad y capacidad de respuesta. Dado que Traffic Manager es un servicio de equilibrio de carga basado en DNS, solo equilibra la carga a nivel de dominio. Por ese motivo, no puede conmutar por error tan rápido como Front Door, debido a los desafíos comunes relacionados con el almacenamiento en caché de DNS y los sistemas que no respetan los TTL de DNS.

Application Gateway

Proporciona un controlador de entrega de aplicaciones (ADC) como un servicio, que ofrece varias capacidades de equilibrio de carga de capa 7. Úselo para optimizar la productividad de la granja de servidores web al descargar la terminación SSL con uso intensivo de la CPU en la puerta de enlace.

Azure Load Balancer

Es un servicio de equilibrio de carga de nivel 4 (entrante y saliente) de alto rendimiento y latencia ultrabaja para todos los protocolos UDP y TCP. Está diseñado para manejar millones de solicitudes por segundo mientras garantiza que su solución esté altamente disponible. Azure Load Balancer tiene redundancia de zona, lo que garantiza una alta disponibilidad en las zonas de disponibilidad.
